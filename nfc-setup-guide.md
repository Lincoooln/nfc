# NFC 标签设置指南

本指南将教您如何将网站链接写入 NFC 标签，让用户通过手机碰触标签即可打开占卜网站。

## 什么是 NFC？

NFC（Near Field Communication，近场通信）是一种短距离无线通信技术。当您用支持 NFC 的手机靠近 NFC 标签时，手机会自动读取标签信息并执行相应操作（如打开网页）。

## 准备工作

### 您需要：

1. **NFC 标签**
   - 推荐类型：NTAG215 或 NTAG216
   - 数量：根据需要购买（通常 10-50 个装）
   - 价格：约 ￥1-3 元/个
   - 购买渠道：淘宝、京东、亚马逊

2. **支持 NFC 的手机**
   - iPhone（iPhone 7 及以上，iOS 13+）
   - Android（大部分中高端手机）

3. **NFC 写入应用**
   - iOS：推荐 "NFC Tools" 或 "NFC TagWriter"
   - Android：推荐 "NFC Tools" 或 "NFC TagWriter by NXP"

4. **已部署的网站链接**
   - 例如：`https://yourusername.github.io/book-of-answers/`

## NFC 标签推荐

### 推荐型号：

| 型号 | 存储空间 | 适用场景 | 价格范围 |
|------|----------|----------|----------|
| NTAG213 | 144 字节 | 短链接 | ￥1-2/个 |
| NTAG215 | 504 字节 | 一般使用（推荐） | ￥2-3/个 |
| NTAG216 | 888 字节 | 长链接/更多数据 | ￥3-5/个 |

### 标签形态：

- **圆形贴纸**：最常见，直径 25-30mm，易于粘贴
- **卡片式**：名片大小，更专业
- **钥匙扣**：便于携带
- **腕带**：可穿戴
- **创意形状**：星星、心形等

**推荐**：选择 NTAG215 圆形贴纸，性价比最高，适合大部分场景。

## 第一步：安装 NFC 应用

### iOS 用户：

1. 打开 App Store
2. 搜索 "NFC Tools"
3. 下载并安装（免费）

### Android 用户：

1. 打开 Google Play 或应用商店
2. 搜索 "NFC Tools"
3. 下载并安装（免费）

## 第二步：准备网站链接

1. 确保您的网站已成功部署（见 `deployment-guide.md`）
2. 复制完整的网站链接，例如：
   ```
   https://yourusername.github.io/book-of-answers/
   ```

### 使用短链接（推荐）

为了节省 NFC 标签空间并便于更换链接，建议使用短链接服务：

**方法 A：使用 Bitly**
1. 访问 [bitly.com](https://bitly.com)
2. 注册免费账号
3. 粘贴您的完整链接
4. 获得短链接，例如：`https://bit.ly/my-answers`

**方法 B：使用其他服务**
- TinyURL: https://tinyurl.com
- Is.gd: https://is.gd
- Rebrandly: https://rebrandly.com（支持自定义域名）

**优势**：
- 链接更短，节省空间
- 可以更新链接指向而不需要重写 NFC 标签
- 可以追踪访问数据

## 第三步：写入 NFC 标签

### 使用 NFC Tools（iOS/Android）

1. **打开 NFC Tools 应用**

2. **点击 "Write"（写入）标签页**

3. **添加记录**
   - 点击 "Add a record"（添加记录）
   - 选择 "URL / URI"
   - 输入您的网站链接
   - 点击 "OK" 或 "确定"

4. **写入标签**
   - 点击底部的 "Write"（写入）按钮
   - 将 NFC 标签靠近手机背面（NFC 天线位置）
   - 保持贴近，直到看到成功提示
   - 听到提示音或看到 "Write complete" 表示成功

5. **测试标签**
   - 用手机再次靠近标签
   - 应该自动弹出网站链接
   - 点击打开，确认网站正常显示

### 使用快捷指令（仅 iOS 14+）

iOS 14 及以上版本可以使用系统内置功能：

1. **打开快捷指令 App**

2. **创建自动化**
   - 点击"自动化"标签页
   - 点击右上角 "+"
   - 选择"创建个人自动化"
   - 选择 "NFC"

3. **扫描标签**
   - 点击 "扫描"
   - 将 NFC 标签靠近手机
   - 为标签命名（如 "占卜答案"）

4. **添加动作**
   - 搜索并添加 "打开 URL"
   - 输入您的网站链接
   - 关闭 "运行前询问"
   - 点击 "完成"

5. **测试**
   - 用手机靠近 NFC 标签
   - 应该自动打开网站

## 第四步：放置和装饰 NFC 标签

### 放置建议：

1. **桌面**：工作桌、床头柜
2. **墙面**：门口、冰箱
3. **物品上**：笔记本、日记本
4. **公共区域**：办公室休息区、创意空间

### 装饰想法：

1. **打印贴纸**
   - 设计一个漂亮的标签封面
   - 覆盖在 NFC 标签上（注意：不要太厚，影响读取）
   - 可以写上 "Tap for Answer" 或 "碰触获取答案"

2. **创意容器**
   - 将 NFC 标签贴在一个装饰盒内
   - 贴在书本封面内侧
   - 嵌入相框背面

3. **多个标签**
   - 在不同位置放置多个标签
   - 都指向同一个网站
   - 增加神秘感和可访问性

## 常见问题

### Q: 我的手机支持 NFC 吗？

**iPhone:**
- iPhone 7 及以上型号支持
- iOS 13+ 可自动读取 NFC（无需打开应用）
- iOS 11-12 需要使用 NFC 应用

**Android:**
- 检查方法：设置 → 连接与分享 → NFC
- 或者：设置 → 更多连接方式 → NFC
- 大部分 2016 年后的中高端手机支持

### Q: NFC 标签无法读取？

**解决方法：**
1. 确保手机 NFC 功能已开启
2. 尝试调整手机位置（NFC 天线通常在手机背部中上方）
3. 移除手机壳（金属壳会干扰信号）
4. 确保标签已正确写入（重新写入一次）
5. 更换标签（可能标签损坏）

### Q: iPhone 读取 NFC 没有反应？

**iOS 13+ 用户：**
- 确保未锁屏（iOS 13-14 需要解锁）
- iOS 14+ 可以在锁屏状态读取
- 靠近标签时，屏幕顶部应显示通知

**iOS 11-12 用户：**
- 必须先打开 NFC Tools 等应用
- 选择 "Read" 功能
- 然后靠近标签

### Q: Android 手机读取后没有打开网页？

**解决方法：**
1. 检查 NFC 设置中是否启用了 "Android Beam" 或类似选项
2. 第一次读取时，选择 "始终使用浏览器打开"
3. 尝试不同的浏览器（Chrome、Firefox 等）

### Q: NFC 标签能读取多少次？

**答：**
- 读取次数：无限次
- 写入次数：通常 10 万次以上
- 寿命：正常使用可达 10 年以上
- 不需要电池或充电

### Q: 可以修改 NFC 标签内容吗？

**答：**
- 是的，可以随时重写
- 使用相同的 NFC 应用
- 选择 "Write"，会覆盖原有内容
- 如果想防止修改，可以在写入时选择 "锁定标签"（不可逆）

### Q: NFC 标签有安全风险吗？

**答：**
- NFC 本身很安全（短距离，难以远程攻击）
- 风险主要来自恶意链接
- 只写入您自己的可信链接
- 不要随意碰触陌生的 NFC 标签
- 建议使用短链接，便于更换和追踪

### Q: 可以在 NFC 标签上添加其他信息吗？

**答：**
可以！使用 NFC Tools，您可以添加：
- 多条记录（链接 + 文本）
- WiFi 连接信息
- 联系信息（vCard）
- 地理位置
- 自定义文本

但对于本项目，建议只存储网站链接，保持简洁。

## 高级技巧

### 1. 批量写入标签

如果您需要写入多个相同内容的标签：

1. 在 NFC Tools 中，点击右上角菜单
2. 选择 "Save tasks"（保存任务）
3. 之后可以快速重复写入

### 2. 添加密码保护

使用 NFC Tools Pro（付费版）：
1. 可以为标签设置密码
2. 只有知道密码的人才能修改
3. 但任何人都能读取（打开网站）

### 3. 统计访问数据

如果使用 Bitly 等短链接服务：
1. 登录 Bitly 账号
2. 查看链接统计
3. 了解 NFC 标签被使用的次数、时间、地点

### 4. 定期更换答案

1. 定期编辑 `index.html` 中的答案列表
2. 提交更新到 GitHub
3. 网站自动更新
4. NFC 标签无需重新写入（因为链接不变）
